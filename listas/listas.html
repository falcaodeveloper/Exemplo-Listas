HTML5
A HTML é uma linguagem de marcação para página web. Não é uma linguagem de programação.
CSS3
As CSS, Folhas de Estilo em Cascata, são responsáveis pelo layout da página web. A aparência da página fica por conta das CSS.

JavaScript é uma linguagem de programação que permite a interação do usuário com a página ou elementos da mesma.
